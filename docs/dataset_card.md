# md-reheader Dataset

Training data for markdown heading level restoration.

## License

This dataset is released under **CC-BY-SA 4.0** because it contains derivative
content from Wikipedia (via euirim/goodwiki, CC-BY-SA 4.0). The GitHub-sourced
portion (via codeparrot/github-code) contains only permissively-licensed files
(MIT, Apache 2.0, BSD, etc.).

## Sources

- **codeparrot/github-code** (Markdown subset) — 100k+ files, permissive licenses
- **euirim/goodwiki** — 44.8k curated Wikipedia "Good" and "Featured" articles (CC-BY-SA 4.0)
